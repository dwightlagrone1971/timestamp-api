API Basejump: Timestamp microservice

User Stories

  • I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: March 1, 2017)
  • If it does, it returns both the Unix timestamp and the natural language form of that date.
  • If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://timestamp-ms.herokuapp.com/January%201,%202016

https://timestamp-ms.herokuapp.com/1451606400000

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
